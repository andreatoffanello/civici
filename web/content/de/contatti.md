## Schreiben Sie uns

Haben Sie Fragen, Vorschläge oder einen Datenfehler gefunden? Wir freuen uns von Ihnen zu hören.

**E-Mail**: info@dovevenezia.com

## Fehler melden

Wenn eine Hausnummer nicht mit dem tatsächlichen Standort übereinstimmt, teilen Sie uns dies bitte per E-Mail mit dem Sestiere, der Hausnummer und der richtigen Position mit.

## Open Source

DoVe ist ein Open-Source-Projekt. Sie können auf GitHub zum Code und zu den Daten beitragen.
