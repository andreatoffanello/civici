## Questions fréquentes

### Comment rechercher un numéro civique ?

Ouvrez l'application, sélectionnez le sestiere (ou l'île) où se trouve l'adresse, puis tapez le numéro civique dans le champ de recherche. L'application affichera les résultats correspondants en temps réel. Appuyez sur un résultat pour voir l'emplacement exact sur la carte.

### Pourquoi "Via Roma 12" ne fonctionne pas ?

DoVe est conçue pour le système unique de numérotation civique de Venise. Le centre historique de Venise n'utilise pas d'adresses basées sur les noms de rues : il utilise sestiere + numéro (ex. "Cannaregio 2345"). Certains noms de lieux comme les calli et les campielli sont indicatifs, mais le numéro du sestiere est l'identifiant principal.

Les îles de la lagune (Murano, Burano, Lido, etc.) et certaines zones utilisent le système classique rue + numéro, que DoVe prend en charge dans leurs sections respectives.

### L'application fonctionne-t-elle sans connexion internet ?

Oui. Les 47 261 numéros civiques de Venise sont intégrés dans l'application. Aucune connexion internet n'est nécessaire pour rechercher un numéro civique. La carte nécessite une connexion internet pour charger les tuiles graphiques d'Apple Maps.

### Un numéro civique est mal positionné. Comment le signaler ?

Envoyez-nous un e-mail à info@dovevenezia.com en indiquant : le sestiere, le numéro civique et la position correcte (vous pouvez joindre une capture d'écran ou indiquer les coordonnées approximatives). Nous vérifierons et corrigerons lors de la prochaine mise à jour.

### Où puis-je trouver le code source ?

DoVe est un projet open source. Vous trouverez le code sur GitHub (lien dans la section À propos).
