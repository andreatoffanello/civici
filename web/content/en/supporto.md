## Frequently Asked Questions

### How do I search for a civic number?

Open the app, select the sestiere (or island) where the address is located, then type the civic number in the search field. The app will show matching results in real time. Tap a result to see the exact location on the map.

### Why doesn't "Via Roma 12" work?

DoVe is designed for Venice's unique civic numbering system. The historic center of Venice does not use street-based addresses: it uses sestiere + number (e.g. "Cannaregio 2345"). Some place names like calli and campielli are indicative, but the sestiere number is the primary identifier.

The lagoon islands (Murano, Burano, Lido, etc.) and some areas use the classic street + number system, which DoVe supports in their respective sections.

### Does the app work without an internet connection?

Yes. All 47,261 Venice civic numbers are bundled in the app. No internet connection is needed to search for a civic number. The map requires an internet connection to load Apple Maps tile graphics.

### A civic number is in the wrong location. How do I report it?

Send us an email at info@dovevenezia.com with: sestiere, civic number, and the correct position (you can attach a screenshot or provide approximate coordinates). We'll verify and fix it in the next update.

### Where can I find the source code?

DoVe is an open-source project. You can find the code on GitHub (link in the About section).

If the app behaves unexpectedly, send us an email with:

1. Your iPhone model and iOS version
2. The DoVe version (visible in Settings → About)
3. A description of the issue and, if possible, steps to reproduce it
